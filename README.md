# Pisteur's Drone - Description package
Pisteur's Drone are ROS2 packages designed for simulating drones in a Gazebo environment. This package provides models and physics properties for drones.

Models from [Gazebo Fuel](https://app.gazebosim.org/OpenRobotics/fuel/models/X3%20UAV).

## Associated packages
- [pisteur_drone_description](https://github.com/pisteuralpin/pisteur_drone_description) (this package)
- [pisteur_drone_gazebo](https://github.com/pisteuralpin/pisteur_drone_gazebo)

## Features
- [x] Drone models with realistic physics properties.
